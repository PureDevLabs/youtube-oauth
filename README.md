# youtube-oauth

To install dependencies:

### via bun
```bash
bun install
```

To run:

```bash
bun run index.js
```

---

### via node / npm

```bash
npm install
```
To run:
```bash
node run index.js
```

This project was created using `bun init` in bun v1.1.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
